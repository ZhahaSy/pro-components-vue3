# ActionDialog 组件

> 点击按钮，打开弹窗

### 用法

```vue
<ActionDialog type="primary">
</ActionDialog>
```

```jsx
<ActionDialog type={primary}>
</ActionDialog>
```

### API

| 属性        | 说明         | 类型                    | 默认值   |
| ----------- | ------------ | ----------------------- | -------- |
| type       | 按钮类型     | string                  | -        |

> size 为 middle 时，宽 3，高 11，为 small 时，宽 2，高 8
